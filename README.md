# DataCo-Supply-Chain-Analysis-Using-Excel
Excel-based supply chain analysis focused on sales, profitability, and delivery performance optimization.
DataCo Supply Chain Analysis
🔍 Overview

An Excel-based supply chain analytics project focused on analyzing sales, profit, delivery performance, and logistics efficiency using e-commerce order data. The project identifies late deliveries, loss-making orders, and high-risk shipping patterns to support data-driven business decisions.

🎯 Business Goals

Evaluate sales and profitability across regions and products

Identify delivery delays and their impact on profit

Analyze shipping modes to improve logistics efficiency

Detect loss-making and high-risk orders

🗂 Dataset

Order-level e-commerce transactions

Sales, profit, discounts, and benefit per order

Shipping performance & delivery status

Time trends and geographic details

🛠 Tools & Techniques

Tool: Microsoft Excel

Techniques: Data cleaning, helper columns, dashboards

Functions Used: SUMIFS, XLOOKUP, Data Validation

📊 Key Metrics

Total Orders: 180,510

Total Revenue: $36.78M

Total Profit: $3.97M

Profit Margin: 11%

✅ Outcome

Delivered actionable insights to reduce delivery delays, improve profitability, and optimize supply chain operations.
